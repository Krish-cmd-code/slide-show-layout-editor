# Customizable Web Layout Editor with Drag‑and‑Drop Slideshow

### Features ###
- drag and reposition UI elements
- toggle customization mode
- save layout positions
- reload the saved layout
- manage image containers and galleries

### Description ###
 Built a web interface that allows users to dynamically rearrange UI components via drag‑and‑drop, toggle customization mode, and persist element positions across sessions.
